Application Description and Purpose

This application is designed to help those not experienced in computer science or web development teach basic web development and computer science in a classroom setting.

Intended Users

Education professionals and their students.

Website Content Overview

The website will provide a web development learning platform for elementary to early school students, covering HTML, CSS, and JavaScript. It includes tests and practices to help solidify knowledge in an approachable way that does not require prior experience from the teacher.

Pages Overview

Actions: Navigate to other pages via navbar.

Purpose: Display class rankings for the student’s class.

Audience: Students and education professionals.

Content: Personal and class completion data, visual representations.

Actions: Select metric via dropdown, display dynamic data.

Purpose: Web development lessons and practices.

Audience: Students.

Content: Lessons, interactive coding practices, tests.

Actions: Submit exercises, validate answers, store progress locally.

Purpose: Administrative page to view student progress.

Audience: Teachers only.

Content: Student progress data, search bar for student lookup.

Actions: Login verification, display dynamic student data visualizations.

Dynamic Functionality

JavaScript will be used to implement:

  • Modals on the introduction page.
  • Slideshow of images from a photo gallery.
  • Dropdowns and search bars on Class Rankings and Admin pages.
  • Loading/hiding lessons, practices, and tests on Lessons page.
